# Placeholder And Random Governance

## Placeholder

- Placeholder 字段必须以 `_placeholder` 结尾。
- Placeholder 字段必须在 `docs/field_registry.md` 中登记。
- Placeholder 字段不得支持 supported claims。

## Random Trace

- Random trace 字段必须以 `_random` 或 `_digest_random` 结尾。
- Random trace 字段必须具备 seed、digest 或可复现说明。
- Random trace 字段不得支持 supported claims。
